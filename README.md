# 代码描述
这个Python程序实现了**外卖评论情感分析系统**（将评论分类为正面/负面），核心功能与工作流程如下：

## 1. 数据预处理
- 加载外卖评论数据集（`waimai_10k.csv`），统计并可视化正面/负面评论的分布情况；
- 对评论文本进行清洗（去除标点符号），通过`jieba`库完成中文分词，并过滤停用词（使用`hit_stopwords.txt`）。

## 2. 特征工程
- 采用TF-IDF算法将文本数据转换为数值型特征；
- 按照8:2的比例将数据集划分为训练集和测试集。

## 3. 模型训练与评估
- 训练并评估多种经典分类算法：逻辑回归、多项式朴素贝叶斯、支持向量机（SVM，含加权版本适配不平衡数据）、随机森林；
- 使用准确率、分类报告、混淆矩阵评估模型效果，并通过热力图可视化混淆矩阵。

## 4. 可视化分析
- 分别为正面、负面评论生成词云图，直观展示不同情感类别的核心关键词。

## 5. 模型保存与推理
- 构建完整机器学习流水线（TF-IDF特征提取+SVM模型），通过`joblib`保存模型用于部署；
- 提供交互式预测函数，支持输入自定义外卖评论，自动预测情感类别（正面/负面），并输出置信度与概率分布。

## 核心用途
自动完成外卖评论的正负情感分类，对比不同分类模型的性能，同时支持对新评论进行实时情感预测，具备可解释性（置信度/概率）。
